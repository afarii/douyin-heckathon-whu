# 06 运营应用与治理

## 1. 运营SOP

### 1.1 周节奏

1. 周一：更新分层与标签，输出本周策略包。
2. 周二到周四：执行分群内容投放与活动任务。
3. 周五：复盘增长、留存、争议案例并修正策略。

### 1.2 人群策略

| 人群 | 核心目标 | 策略 |
|---|---|---|
| TI-S/TI-A | 稳定高质量供给 | 提供创作资源位、导师权益 |
| TI-B | 推进到A层 | 给到专题任务与成长激励 |
| TI-C/TI-D | 提升参与深度 | 新手任务、低门槛活动 |
| TI-F | 激活兴趣 | 轻解释内容与低干扰触达 |

## 2. 分发策略

- 拉新池：优先高感染力内容。
- 留存池：优先高共鸣内容。
- 活动池：优先高裂变内容。
- 沉淀池：优先高归属内容。

## 3. 活动模板

| 活动类型 | 目标 | 指标 |
|---|---|---|
| 二创接力赛 | 提升裂变 | 二创数、分享率 |
| 梗史讲解周 | 提升共鸣 | 长评率、收藏率 |
| 镇民勋章季 | 提升归属 | 回访率、活跃天数 |

## 4. 风险治理

- 内容风险：争议词、攻击性引导、不当行为暗示。
- 社区风险：标签歧视、圈层排斥、恶意引战。
- 商业风险：过度联名导致文化稀释与反感。

## 5. 合规与伦理

- 对外展示必须附“趣味框架，非心理诊断”。
- 未成年人策略：降低刺激性内容推荐权重。
- 提供透明申诉入口，确保标签可纠偏。

## 6. 看板指标（建议）

| 模块 | 指标 |
|---|---|
| 增长 | 新增用户、拉新转化率 |
| 质量 | 完播率、收藏率、长评率 |
| 传播 | 分享率、二创率、跨圈流量占比 |
| 沉淀 | 7日回访率、活跃天数、称号留存 |
| 治理 | 违规命中率、申诉通过率 |

## 7. 实施里程碑

- 第1周：埋点与数据表上线。
- 第2周：TI评分与MBTI标签首版上线。
- 第3周：称号机制灰度发布（10%人群）。
- 第4周：全量发布并完成首轮复盘。
