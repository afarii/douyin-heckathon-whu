# 00 总览与范围

## 1. 体系目标

本体系用于统一“内容评估 + 用户分层 + 运营决策”三条链路：

1. 识别哈吉米相关内容的文化价值与传播潜力。
2. 识别用户参与深度与互动偏好。
3. 将评估结果转化为可执行运营动作（推荐、活动、激励、风控）。

## 2. 适用范围

- 内容对象：音频、短视频、话题页、二创合集、联名活动。
- 用户对象：游客、轻度参与者、核心创作者、社群管理者。
- 业务对象：内容分发、社群运营、商业合作、治理合规。

## 3. 非适用声明

- 本体系不是心理学诊断工具。
- 本体系不是法律裁定工具。
- 任何标签均不得用于人身攻击与歧视。

## 4. 核心术语

| 术语 | 定义 |
|---|---|
| TI等级 | 内容或用户在体系中的综合影响力等级（S/A/B/C/D/F） |
| MBTI类型 | 基于行为问卷与互动日志映射出的16型偏好标签 |
| 称号 | 结合等级、贡献与文化语境生成的荣誉标签 |
| 分层 | 基于阈值规则对内容/用户进行运营分组 |
| 稀释风险 | 商业化后文化语境丢失与社区反感风险 |

## 5. 治理原则

- 可解释：每个标签均能追溯到规则与字段。
- 可复算：同一输入在同一版本下输出一致。
- 可审计：支持抽样复核、人工申诉与回滚。
- 可迭代：按月校准阈值，按季度校验权重。

## 6. 版本策略

- 月更：词库、热点话题、标签白名单。
- 季更：评分权重、阈值区间、升降级参数。
- 半年：全链路审计（数据质量、误分率、运营收益）。
