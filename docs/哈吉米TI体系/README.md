## 快速落地顺序

1. 先按 `05` 完成数据埋点与每日批处理。
2. 再按 `01 + 02 + 03` 产出首版用户标签。
3. 按 `04` 配置称号发放与回收。
4. 按 `06` 建立周报、月度复盘与异常处理流程。

## 最低可用版本（MVP）

- 数据维度：播放、点赞、评论、分享、收藏、完播、二创。
- 评分频率：内容日更、用户周更。
- 输出结果：`ti_level`、`mbti_type`、`title_level`、`ops_segment` 四类标签。
